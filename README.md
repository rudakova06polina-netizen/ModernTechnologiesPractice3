## Требования

- JDK 21+
- IntelliJ IDEA Community
---

## Как работать с лабораторной

1. Откройте нужный класс по заданию (см. `aliebraheem-fun.github.io/Modern-Programming-Technologies/#/practice3`).
2. Прочитайте Javadoc и блоки `Подсказка`.
3. Реализуйте участки внутри TODO-маркеров:
   - `// ▼ ВАШ КОД ЗДЕСЬ ▼`
   - `// ▲ КОНЕЦ ВАШЕГО КОДА ▲`
4. Запустите `main()` и сравните результат с ожидаемым.
5. Зафиксируйте прогресс коммитом.

---

## Что сдавать

1. Все `.java` файлы в `part1` ... `part5` с заполненными TODO.
2. Ответы на контрольные вопросы в `answers/part6_questions.md`.
